# International Student Mental Health Analysis

## Project Overview

This project analyzes survey data from a Japanese international university to explore whether studying abroad may be associated with student mental health outcomes. The analysis focuses on international student status, social connectedness, acculturative stress, length of stay, and depression-related measures.

## Research Question

Does attending university in a different country appear to affect student mental health, and what factors are associated with depression-related outcomes among international students?

## Tools Used

- PostgreSQL
- SQL
- DataCamp DataLab
- Jupyter Notebook
- GitHub

## Dataset

The dataset includes student survey responses related to demographics, international student status, social connectedness, acculturative stress, and depression-related measures.

## Analysis Performed

- Explored student records and key mental health variables
- Compared international and domestic student groups
- Analyzed the relationship between length of stay and depression-related outcomes
- Used SQL queries to filter, group, and summarize student survey data
- Interpreted results in the context of student well-being and support services

## Key Skills Demonstrated

- SQL querying
- Data filtering
- Aggregation and grouping
- Exploratory data analysis
- Analytical interpretation
- Communicating findings clearly

## Files

- international_student_mental_health_analysis.ipynb — Full project notebook

## Summary

This project demonstrates how SQL can be used to explore survey data, identify patterns, and generate insights that may help universities better understand student mental health challenges, especially among international students.
