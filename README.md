### 2. International Student Mental Health Analysis

**Overview:**  
Used SQL to analyze student survey data from a Japanese international university to explore whether studying abroad may be associated with mental health outcomes. The project focuses on international student status, social connectedness, acculturative stress, length of stay, and depression-related measures.

**Tools Used:**  
PostgreSQL, SQL, DataCamp DataLab, Jupyter Notebook

**Key Skills Demonstrated:**  
- SQL querying  
- Data filtering and aggregation  
- Exploratory analysis  
- Interpreting survey data  
- Translating data findings into insights  

**Project Files:**  
- Notebook: international-student-mental-health-sql/international_student_mental_health_analysis.ipynb  
- Dataset: international-student-mental-health-sql/data/students.csv
