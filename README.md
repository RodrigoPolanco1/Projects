# Customer Sentiment Analysis & Insights for Product Strategy – Beats by Dre

## 📌 Overview
Analyzed customer reviews to uncover sentiment trends and provide product strategy recommendations.

## 📊 Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- NLP / Sentiment Analysis

## 🔍 Key Insights
- Positive sentiment driven by sound quality
- Negative sentiment linked to battery life issues
- Competitors outperform in comfort
